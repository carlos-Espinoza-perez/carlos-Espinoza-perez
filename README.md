# Carlos Espinoza

Desarrollador de software enfocado en construir soluciones reales para negocios, combinando desarrollo web, aplicaciones móviles, automatización e integración de inteligencia artificial.

Actualmente estudio Ingeniería en Mecatrónica y Sistemas de Control, lo que me permite unir la lógica del software con procesos operativos, sistemas administrativos y soluciones aplicadas a entornos reales.

Mi enfoque principal está en crear sistemas que ayuden a organizar, automatizar y mejorar procesos dentro de empresas.

## Sobre mí

Soy desarrollador con experiencia práctica en proyectos web, móviles y sistemas administrativos. Me interesa especialmente el desarrollo de plataformas para gestión de inventario, planificación, control de procesos, automatización y herramientas con IA aplicada.

Me gusta trabajar en soluciones que no solo funcionen técnicamente, sino que también sean útiles, mantenibles y entendibles para las personas que las usan.

Actualmente estoy fortaleciendo mi perfil en:

- Desarrollo frontend moderno.
- Backend con APIs y bases de datos.
- Aplicaciones móviles.
- Automatización con inteligencia artificial.
- Sistemas empresariales y herramientas internas.
